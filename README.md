# Bibliobattle Survey — Companion Site

This is the public companion website for an in-preparation review paper on **Bibliobattle** (知的書評合戦),
a social book review game invented in Japan in 2007 and now widely adopted as a reading-promotion
practice across Japanese universities and libraries.

The site is hosted via GitHub Pages at:
**https://tanichu.github.io/bibliobattle-survey-companion/**

## Purpose

Most academic literature on Bibliobattle is in Japanese. This site serves as a bilingual bridge,
providing English summaries, original Japanese references, dissemination statistics, and a
historical timeline so that international readers can navigate and verify the sources cited
in the upcoming review paper.

## Site Structure

- `index.html` — Project overview and navigation
- `references.html` — Bilingual list of academic references (~45 entries)
- `stats.html` — Dissemination statistics with sources
- `timeline.html` — 18-year chronology (2007–2026)
- `resources-jp.html` — Curated guide to Japanese-language resources
- `style.css` — Shared stylesheet

## Status

**Phase 1 (May 2026)** — Initial public release with basic content.

The companion review paper is being prepared for submission to JASIST (Journal of the Association
for Information Science and Technology, Wiley). Site content will be expanded in phases as the
manuscript progresses.
